# diveboard-v2

## Frontend

Before start <br>

Reveal secrets

`git secret reveal`

Install dependencies

`cd frontend/ && npm i`

### Development

To start dev server on [localhost](http://localhost:3000/)

`npm run dev`


### Production

**Build**

`npm run build`

**Start production server**

`npm run start`

## Tests

Run tests

`npm run test`
